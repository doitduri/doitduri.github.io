## 5. when expression

```
fun describe(obj: Any): String = 
	when (obj) {
			1 -> "one"
			"Hello" -> "Greeting"
			is Long -> "Long"
			!is String -> "Not a string"
			else       -> "Unknown"
}
```

5) Overriding 규칙
같은 멤버에 대한 중복된 구현을 상속 받은 경우
상속 받은 클래스는 해당 멤버를 오버라이딩하고, 자체 구현을 제공해야함
super + <className>을 통해 상위 클래스 호출 가능
```
interface B {
	fun f() {println("B")}
	...
}

open class A {
	open fun f() {println("A")}
	...
}

class C(): A(), B{
	override fun f() {
		super<A>.f() // call to A.f()
		super<B>.f() // call to B.f()
	}
}
```
